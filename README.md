# GKS
`with Golang 1.7+`

## Get Dependencies
$ go get github.com/astaxie/beego<br/>
$ go get github.com/beego/bee<br/>

## Update Dependencies
$ go get -u github.com/astaxie/beego<br/>
$ go get -u github.com/beego/bee<br/>

## Build/Run
$ go build -o gks main.go<br/>
$ ./gks<br/>
or<br/>
bee run<br/>
